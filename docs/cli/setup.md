# `tigerpaw setup`

Initialize `~/.tigerpaw/tigerpaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
tigerpaw setup
tigerpaw setup --workspace ~/.tigerpaw/workspace
```

To run the wizard via setup:

```bash
tigerpaw setup --wizard
```
